# edeltraud

The IRC bot of the #cahoots.pw channel on irc.freenode.net

## Installation

### Building the Docker image

```sh
docker build -t cahoots.pw/edeltraud .
```

### Starting the bot

```sh
docker run -d -p 9091:9091 --name edeltraud cahoots.pw/edeltraud
```
